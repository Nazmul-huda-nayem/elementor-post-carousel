== Description ==

**Post Carousel for Elementor** is a powerful plugin that enhances your Elementor page builder by adding stylish and customizable blog post carousels. With over **40+ unique and responsive preset designs**, this plugin is ideal for showcasing your latest blog posts, news, articles, or any custom post type in a modern carousel layout.

Whether you're creating a blog, magazine, portfolio, or business website, this addon will help you attractively present your content and increase user engagement.

🎯 **Key Features:**

- 🆓 First **2 preset carousel styles are completely FREE**
- 🔐 **40+ modern, premium designs** for [pro](https://codecanyon.net/item/creative-post-carousel-addon-for-elementor/41061763) users
- 📱 Fully responsive and mobile-friendly
- 🎨 Extensive customization options
- 🔁 Autoplay, loop, navigation arrows, and dots
- 📰 Supports default WordPress posts and custom post types
- 🛠 Built using Elementor’s native controls – no learning curve
- ⚡ Fast and lightweight
- 💻 Developer-friendly code structure

🧩 **Dependencies:**

- [Owl Carousel](https://owlcarousel2.github.io/OwlCarousel2/) – for smooth and responsive carousel functionality
- [Font Awesome](https://fontawesome.com/) – for icons (arrows, UI elements, etc.)

🔥 Whether you're a beginner or a pro, creating attractive post carousels in Elementor has never been easier!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/post-carousel-for-elementor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Edit any page with Elementor and search for **“Post Carousel”** in the Elementor widget panel.
4. Drag and drop the widget and start customizing your carousel.
